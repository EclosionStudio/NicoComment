## NicoNicoNi 社会化评论增强系统

一个正在开发的社会化评论增强插件。

React-Redux-Thunk 技术栈验证。

后端开发时采用Nodejs， 将来可以让用户选择使用PHP/Python等等任何一种后端语言。

## 功能：

随时随地获取你的文章评论。
